# RelayStack Remotion Video

60-90s RelayStack introduction video.

```bash
pnpm install
pnpm run studio
pnpm run render
```

`npm run studio` / `npm run render` also works after installing with npm.

```text
public/voiceover/relaystack-intro/voiceover.wav
```
